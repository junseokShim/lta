# 👋 Python Greeting CLI

사용자로부터 이름을 입력받아 맞춤형 인사말을 출력하는 간단하고 실행 가능한 CLI(Command Line Interface) 도구입니다. 이 프로젝트는 파이썬의 `argparse` 라이브러리를 활용하여 명령행 인자 전달 방식과 대화형 입력 방식을 모두 지원합니다.

## 🎯 프로젝트 목적
- 명령행 인자(Command Line Arguments)를 처리하는 CLI 애플로케이션의 기본 구조 학습
- 사용자 입력에 따른 예외 처리 및 인터랙티브한 UX 구현
- 자동화된 기능 검증을 위한 Smoke Test 기능 포함

## ✨ 주요 기능 (Features)
- 🚀 **CLI 인자 지원**: `--name` 옵션을 사용하여 실행 시 즉시 이름을 지정할 수 있습니다.
- 💬 **인터랙티브 모드**: 인자를 전달하지 않을 경우, 사용자에게 직접 이름을 입력받는 대화형 모드로 동작합니다.
- 🧪 **Smoke Test 기능**: `--smoke-test` 플래그를 통해 애플리케이션의 핵심 로직이 정상 작동하는지 빠르게 확인할 수 있습니다.
