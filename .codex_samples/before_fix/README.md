# 🤖 Simple Python Greeter CLI

이 프로젝트는 사용자로부터 이름을 입력받아 인사말을 출력하는 간단하고 실행 가능한 Command Line Interface (CLI) 애플리케이션입니다.

## ✨ 프로젝트 소개 및 목적

이 애플리케이션은 파이썬을 사용하여 사용자와 상호작용하는 기본적인 CLI 도구를 만드는 것을 목표로 합니다. 사용자가 입력한 이름을 받아 친절한 인사말을 제공하는 기능을 구현하며, 직접 실행할 수 있는 엔트리포인트(Entrypoint)를 제공하여 사용 편의성을 높입니다.

## 🚀 주요 기능 (Features)

*   **이름 입력 기능:** 사용자로부터 이름을 입력받습니다.
*   **인사말 생성:** 입력받은 이름을 사용하여 맞춤형 인사말을 출력합니다.
*   **실행 가능한 CLI:** 스크립트를 직접 실행하여 즉시 결과를 확인할 수 있습니다.
*   **간결한 구현:** 파이썬의 기본 기능을 활용하여 최소한의 코드로 목표를 달성합니다.

## 🛠️ 설치 방법 (Installation)

이 프로젝트는 별도의 복잡한 외부 라이브러리 설치 없이 순수 파이썬 환경에서 바로 실행할 수 있습니다.

1.  **코드 다운로드:** 프로젝트 폴더 내의 스크립트를 확인합니다.
2.  **실행 준비:** 필요한 경우, 프로젝트 디렉토리로 이동합니다.

*(참고: 만약 `setup.py` 파일이 존재한다면, 패키징 및 설치를 위해 다음 명령어를 사용할 수 있습니다.)*

```bash
# 가상 환경 설정 (권장)
python -m venv venv
source venv/bin/activate  # Linux/macOS
.\venv\Scripts\activate  # Windows

# 설치 (setup.py가 존재할 경우)
pip install .
```
